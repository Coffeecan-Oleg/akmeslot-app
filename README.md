# АкмеСлот (AkmeSlot)

Система бронирования корпоративных ресурсов для Битрикс24.

## Описание

АкмеСлот позволяет сотрудникам компании бронировать:
- Переговорные комнаты
- Рабочие места
- Служебные автомобили
- Оборудование

## Функционал

- 🔐 Авторизация через Битрикс24
- 📋 Просмотр доступных ресурсов
- 📅 Бронирование с выбором даты и времени
- 📊 Управление своими бронированиями
- ❌ Отмена бронирований
- 🔍 Фильтрация по категориям

## Технологии

- Node.js + Express
- Vanilla JavaScript (ES6+)
- REST API
- Bitrix24 REST API

## Установка

```bash
npm install
npm start
```

## API Endpoints

### Ресурсы
- `GET /api/resources` — список ресурсов
- `GET /api/resources/:id` — информация о ресурсе
- `POST /api/resources` — создать ресурс
- `PATCH /api/resources/:id` — обновить ресурс
- `DELETE /api/resources/:id` — удалить ресурс

### Бронирования
- `GET /api/bookings` — список бронирований
- `POST /api/bookings` — создать бронирование
- `PATCH /api/bookings/:id` — обновить бронирование
- `DELETE /api/bookings/:id` — отменить бронирование

### Пользователи
- `GET /api/me` — текущий пользователь (через Bitrix24 auth)
- `GET /api/users` — список пользователей
- `GET /api/users/:id` — информация о пользователе

## Интеграция с Битрикс24

1. Создайте приложение в Битрикс24 (Разработчикам → Интеграции)
2. Укажите URL обработчика: `https://your-domain.com/`
3. Добавьте scope: `user`
4. Установите приложение
